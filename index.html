설치 가이드

단계별 설치 방법

어떤 에이전트를 설치하시겠어요? 탭을 선택하면 해당 에이전트의 설치 순서를 확인할 수 있습니다.

1

Node.js 설치 (LTS 버전)

Claude Code는 Node.js 18 이상이 필요합니다. 아래 명령어로 버전을 확인하세요. 없다면 nodejs.org에서 LTS 버전을 다운로드하세요.

terminal
node --version
2

Claude Code 설치

npm을 사용해 Claude Code를 전역으로 설치합니다.

terminal
npm install -g @anthropic-ai/claude-code
3

작업 폴더로 이동 후 실행

작업할 프로젝트 폴더로 이동한 뒤 claude 명령어로 시작합니다.

terminal
cd 내-프로젝트-폴더 claude
4

로그인 인증

최초 실행 시 Claude.ai 계정 또는 Anthropic API 키로 인증합니다. 브라우저가 자동으로 열립니다.

Pro/Max 구독자: Claude.ai 계정으로 로그인하면 구독 사용량이 적용됩니다. API 키를 별도로 설정하면 API 과금이 적용됩니다.
5

(선택) CLAUDE.md 파일로 맞춤 설정

프로젝트 루트에 CLAUDE.md 파일을 만들면 Claude Code가 항상 이 지시사항을 참고합니다.

CLAUDE.md 예시
# 프로젝트 규칙 - 한국어로 응답해줘 - 코드 주석은 영어로 - 컴포넌트는 함수형으로 작성 - 커밋 메시지는 Conventional Commits 형식 사용
1

Codex CLI도 Node.js 22 이상을 권장합니다.

terminal
node --version

Codex CLI 설치

OpenAI API 키 설정

platform.openai.com에서 API 키를 발급받은 뒤 환경변수로 등록합니다.

macOS / Linux
export OPENAI_API_KEY="sk-..." # 영구 설정: ~/.zshrc 또는 ~/.bashrc에 위 줄 추가 후 source ~/.zshrc
Windows (PowerShell)
$env:OPENAI_API_KEY = "sk-..."

핵심 개념

알아두면 좋은 용어들

AI 에이전트를 사용하기 전에 알아두면 좋은 기본 개념들을 정리했습니다.

💻

CLI (Command Line Interface)

그래픽 UI 없이 텍스트 명령어로 컴퓨터를 제어하는 방식입니다. 터미널(Terminal)이나 명령 프롬프트(CMD)에서 작동합니다. AI 에이전트들이 CLI 방식으로 제공되는 이유는 파일 시스템과 시스템 명령어에 직접 접근할 수 있어 자동화에 유리하기 때문입니다.

🔌

API (Application Programming Interface)

프로그램 간 통신을 위한 표준화된 연결 통로입니다. AI 에이전트들은 모두 API를 통해 AI 모델과 통신합니다. API 키는 이 통신의 인증 수단으로, 사용량에 따라 요금이 부과됩니다.

🔄

Agentic Loop (자율 루프)

AI가 목표를 달성할 때까지 스스로 계획 → 실행 → 검토를 반복하는 방식입니다. Claude Code의 경우: 파일 읽기 → 코드 작성 → 명령 실행 → 결과 확인 → 수정 과정을 반복합니다. 사람이 중간에 개입하지 않아도 복잡한 작업을 완수합니다.

🧩

MCP (Model Context Protocol)

AI가 외부 도구 및 데이터 소스와 안전하게 연결하기 위한 표준 프로토콜입니다. Claude Code에서 지원하며, 웹 검색, 데이터베이스, Slack, GitHub 등 다양한 서비스와 연동할 수 있습니다. 플러그인 시스템처럼 AI의 능력을 무한히 확장할 수 있습니다.

📝

토큰 (Token)

AI가 텍스트를 처리하는 단위입니다. 영어는 약 4글자, 한국어는 약 1~2글자가 1토큰에 해당합니다. API 요금은 입력 토큰(내가 보낸 텍스트)과 출력 토큰(AI가 생성한 텍스트)을 합산해 계산됩니다. 1M(100만) 토큰당 가격으로 표시됩니다.

🧠

컨텍스트 창 (Context Window)

AI가 한 번의 대화에서 기억할 수 있는 정보의 최대 크기입니다. 클수록 긴 파일이나 많은 대화를 처리할 수 있습니다. Gemini 2.5 Pro는 100만 토큰(약 750만 단어)으로 세 에이전트 중 가장 큽니다.

📁

CLAUDE.md

Claude Code에 특화된 프로젝트 설정 파일입니다. 이 파일에 지시사항을 적어두면 Claude Code가 매번 이를 참고해 일관된 방식으로 작업합니다. 언어 설정, 코딩 스타일, 절대 변경하지 말아야 할 내용 등을 정의할 수 있습니다.

🔒

샌드박스 (Sandbox)

격리된 실행 환경입니다. Codex CLI가 채택한 방식으로, AI가 실행하는 코드나 명령어가 실제 시스템에 영향을 주지 않도록 별도 공간에서 동작합니다. 보안성은 높지만 실제 파일 시스템 접근이 제한됩니다.
